# 자료구조
* 효율적인 접근 및 수정을 가능하게 하는 자료의 집합
* 원소 사이의 관계가 논리적으로 정의된 일정한 규칙에 의해 나열됨

## 선형 자료구조
* 배열(Array), 연결 리스트(Linked List), 스택(Stack), 큐(Queue)

## 비선형 자료구조
* 트리(Tree), 그래프(Graph)
